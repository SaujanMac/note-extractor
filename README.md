
Note Extractor
==============

What this is
------------
A tiny project that extracts structured notes using a local LLM (Phi-3 Mini) and shows results in your browser.

Quick start
-----------
- See setup and model instructions in `INSTRUCTIONS.md`.
- Open `index.html` in your browser, or run a local server:

```bash
python3 -m http.server 8000
```

What this uses
--------------
- Local model: Phi-3 Mini via `llama.cpp` (see `INSTRUCTIONS.md` for download and server steps).
- Frontend: `index.html` — a simple extraction UI.

Files
-----
- `index.html` — browser UI
- `INSTRUCTIONS.md` — detailed setup and run steps for Phi-3 Mini and llama.cpp
